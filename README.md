A markdown parser for React. Implementing the syntax defined at https://www.markdownguide.org/basic-syntax/

- [X] Headings
- [X] Paragraphs
- [X] Line breaks
- [ ] Emphasis
  - [X] Bold text
  - [x] Italic text
  - [X] Bold and italic text
  - [ ] Allow escaping of \* \*\* \_ and \_\_
- [ ] Blockquotes
- [ ] Lists
  - [X] Unordered lists
  - [ ] Ordered lists
  - [ ] Indented blocks and sub lists
- [ ] Code
- [ ] Horizontal rules
- [ ] Links
- [ ] Images
- [ ] Escaping characters
- [ ] HTML
  - [X] \<br\> tags supported
