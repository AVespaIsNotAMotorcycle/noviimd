A markdown parser for React. Implementing the syntax defined at https://www.markdownguide.org/basic-syntax/

- [X] Headings
- [X] Paragraphs
- [X] Line breaks
- [X] Emphasis
  - [X] Bold text
  - [x] Italic text
  - [X] Bold and italic text
- [ ] Blockquotes
- [ ] Lists
- [ ] Code
- [ ] Horizontal rules
- [ ] Links
- [ ] Images
- [ ] Escaping characters
- [ ] HTML
  - [X] \<br\> tags supported
